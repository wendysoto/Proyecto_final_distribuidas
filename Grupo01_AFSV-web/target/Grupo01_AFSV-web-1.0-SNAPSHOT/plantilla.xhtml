Aplicación J2EE

Desarrollo de una aplicación empresarial J2EE, en base a la administración de una tienda la cual permite su manejo de las entidades:

  • Cargo
  • Estado Civil
  • Empleado
  • Proveedor

Esta aplicación ha sido subidad a la IaaS Digital Ocean para poder ser visualizada desde una dirección web en cualquier navegador web.
